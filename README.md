# Neuraluxe-AI Lite - Premium Only

**Neuraluxe-AI Lite** is a lightweight, modular AI assistant designed for premium users. It integrates multiple tools in a compact, fast, and fully online-aware environment.

---

## Features

- **Premium Modules Only** – All tabs require subscription access.
- **Lightweight & Modular** – Uses JSON for storage; minimal dependencies.
- **Chat Functionality** – Full chat history, analytics, and daily usage tracking.
- **Mini Games** – Integrated mini-games tab.
- **Web Search & Knowledge Base** – Online sources for news, jokes, quotes, Wikipedia, and more.
- **Voice Assistant** – Polite male and cheerful female options.
- **Subscription Management** – Auto-unlock for payments via Opay.
- **Analytics** – Mini analytics per user: total chats, sessions, daily usage.
- **Novel Hub & Assets Hub** – Access novels and other content.
- **Full Offline Safety** – Works with cached JSON data if network is unavailable.
